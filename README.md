# Portfolio Website - Pablo Fernando Rincón Díaz

Senior QA Portfolio website build with semantic HTML5 and CSS3.

## Project Structure
- `index.html`: Main portfolio content.
- `styles.css`: Stylesheet using a "Slate & Royal Blue" theme.

## Deployment
1. Upload `index.html`, `styles.css`, and your `resume.pdf` to any static site host (GitHub Pages, Vercel, Netlify).
2. Ensure `resume.pdf` is placed in the root directory.

## Local Development
To view locally:
1. Open `index.html` in any modern web browser.
